# 音游猜曲名刮刮乐

## 曲库

目前支持的音游：Arcaea、Phigros、Orzmic

- 曲库使用的Arcaea版本：v4.5.3
- 曲库使用的Phiros版本：3.1.1.1
- 曲库使用的Orzmic版本：2.18.21

## 命令列表

|          命令         |           功能           |
|:---------------------:|:------------------------:|
|        通用命令       |                          |
|     h \| help \| ?    |         显示帮助         |
|       e \| exit       |           退出           |
|  v \| ver \| version  | 显示曲库使用的音游版本号 |
|       一次性命令      |                          |
|    (u \| use) <num\>   |      选择使用的曲库      |
|     游戏中可用命令    |                          |
|       s \| show       |         显示题目         |
|   (o \| open) <char\>  |        刮开某字符        |
|  (c \| correct) <num\> |      将某题全部刮开      |
| (d \| decrease) <num\> |     减少可用刮开次数     |
| (i \| increase) <num\> |     增加可用刮开次数     |
